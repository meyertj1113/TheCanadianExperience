Detailed Description

Class that represents a machine.

This uses a standin class to provide a way to develop the adapter class first if you so choose.

You are not allowed to change this class in any way!

◆ mStandin

std::shared_ptr<MachineStandin> Machine::mStandin
private

A stand-in machine class.

This draws simple stand-in graphics so you can see where the machine will be and what its state is. This makes it easy to develop code that uses the machine before you have the machine working.

And no, you may not change this to point to your machine implementation!
